# Task 26 - Skip links for main content

Source: ideas/architectural-review-2026/10-accessibility.md

Goal
- Add skip links to jump past the navbar.

Scope
- Add a skip link after body start in all HTML pages.
- Add shared CSS for focus styling.
- Ensure main content has a matching id.

Acceptance criteria
- Keyboard users can tab to a visible skip link.
- The link moves focus to main content.
