# Task 34 - Offline indicator

Source: ideas/architectural-review-2026/13-quick-wins.md

Goal

- Show a small offline banner when the user loses connectivity.

Scope

- Add online and offline listeners in scripts/page-init.js.
- Add a minimal banner style in styles.css.

Acceptance criteria

- Banner appears when offline and disappears when online.
- Banner does not overlap critical UI elements.
